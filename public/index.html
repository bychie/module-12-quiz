What’s Wrong With It

  • Overwhelming, clashing colour blocks (magenta, cyan, yellow, green, etc.).
  • Inconsistent and unreadable typography (Comic Sans, Papyrus, weird sizes).
  • No proper layout or spacing; elements overlap or feel random.
  • No responsive design; fixed large dimensions and non-flexible areas.

Improvement Plan

My plan to fix this UI:
• Establish a simple colour palette (2-3 colours).
• Use a clean, modern font and consistent font sizes.
• Use a proper grid layout with spacing and alignment.
• Remove excessive borders and strokes.
• Ensure the design is responsive and accessible (mobile friendly, good contrast).
